## 🚀 快速开始

### 1. 环境搭建
```bash
# 克隆项目
git clone <repository-url>
cd learn_n8n

# 启动n8n（Docker方式）
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n
```

### 2. 访问n8n
打开浏览器访问：http://localhost:5678

### 3. 开始学习
按照培训计划顺序学习，每天完成相应的实操练习。

## 🔧 常见问题

### Q: 如何解决Docker端口冲突？
A: 修改端口映射：`-p 5679:5678`

### Q: API调用失败怎么办？
A: 检查网络连接、API Key、请求格式

### Q: 工作流执行缓慢？
A: 检查节点配置、网络延迟、数据量大小

### Q: 如何备份工作流？
A: 使用n8n的导出功能或数据库备份
